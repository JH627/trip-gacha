## 🗃 ERD

- ERD 설계 툴: [ERDCloud](https://www.erdcloud.com/d/9xCNb6AfNncS8ifCy)

![ERD](./img/erd.png)

주요 테이블 설명:

- **users**: 사용자 계정 정보
- **destinations**: 목적지 정보 (서울, 부산 등)
- **boards**: 게시글 정보
- **spots**: 관광지 정보
- **trip_schedule**: 여행 일정 정보
